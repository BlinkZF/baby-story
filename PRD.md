**状态**：草稿

## 一、产品概述

### 1.1 产品定位

「宝宝胎教」是一款面向孕期家庭的 AI 胎教陪伴应用。核心价值主张：

> **用爸爸妈妈自己的声音，为宝宝讲故事。**

通过采集父母声音样本，利用 AI 声音克隆技术，将优质胎教内容（故事、儿歌、冥想音频）转化为高度还原父母声线的语音，让胎儿在温暖熟悉的声音陪伴下健康成长。

### 1.2 目标用户

| 用户群体 | 描述 |
|---|---|
| 准妈妈 | 孕 4~9 个月，主要使用者 |
| 准爸爸 | 参与录制，增强陪伴感 |
| 二孩/多孩家庭 | 已有大宝的声音素材，复用意愿高 |

### 1.3 核心价值

1. **情感连接**：父母声音让胎儿更早感知家庭温暖
2. **降低门槛**：不需要父母有专业朗读能力，AI 自动润色
3. **内容丰富**：平台提供海量优质胎教脚本，持续更新
4. **记录成长**：声音档案沉淀为家庭珍贵记忆

### 2.2 核心功能详述

#### F1. 声音采集与克隆

**目标**：用最低的操作成本，采集到足够高质量的声音样本，训练出还原度高的专属声音模型。

| 字段 | 说明 |
|---|---|
| 优先级 | P0（核心差异化功能） |
| 涉及用户 | 爸爸、妈妈 |

**用户流程**：

```
进入「声音采集」→ 阅读录制须知（安静环境、距麦 15cm）
→ 分段朗读引导文本（共 30~50 句，约 5 分钟）
→ 系统实时检测音量 / 噪声 / 清晰度
→ 不合格段落提示重录
→ 全部通过 → 提交后台训练（预计 5~10 分钟）
→ 训练完成推送通知 → 试听确认 → 声音模型保存
```

**功能细节**：

- 引导文本覆盖汉语常见音节（声母、韵母全覆盖），确保模型泛化能力
- 实时波形可视化，辅助用户判断录音效果
- 支持逐句重录，无需全部重来
- 训练完成后提供 3 句试听样本（从胎教故事中抽取）
- 声音模型本地加密存储 + 云端备份
- 每位家长最多保存 3 个声音版本（可更新迭代）

**验收标准**：

- MOS（Mean Opinion Score）主观评测 ≥ 3.8（满分 5）
- 声音相似度评测（余弦相似度）≥ 0.85
- 单次完整录制耗时 ≤ 8 分钟

---

#### F2. 胎教内容库

**目标**：提供丰富、专业、分孕周的胎教内容脚本。

| 分类 | 初期内容量 | 更新频率 |
|---|---|---|
| 睡前故事 | 50 篇 | 每周 3 篇 |
| 经典童话 | 30 篇 | 每月 10 篇 |
| 国学启蒙（三字经等） | 10 篇 | 每月 5 篇 |
| 科普小知识 | 20 篇 | 每周 2 篇 |
| 儿歌童谣 | 40 首 | 每周 5 首 |
| 冥想 & 放松引导 | 15 篇 | 每月 5 篇 |

**内容标签体系**：

- 适用孕周（如「孕 16~20 周」）
- 时长（3 分钟 / 5 分钟 / 10 分钟）
- 情绪标签（平静、欢快、温情、睡前）
- 朗读难度（易 / 中 / 难，用于指导用户录音时选择）

---

#### F3. AI 语音合成播放

**目标**：将文字内容合成为父母声线的高质量音频，供随时播放。

**用户流程**：

```
选择一篇内容 → 选择声音（爸爸 / 妈妈 / 系统声音）
→ 点击「生成专属音频」→ 在线合成（约 10~30 秒）
→ 播放预览 → 满意则保存 / 直接播放
```

**播放器功能**：

- 胎教模式：屏幕变暗，大字体显示内容文字（辅助跟读）
- 速度调节：0.8x / 1.0x / 1.2x
- 背景音乐叠加（轻音乐、自然音效）
- 定时关闭（15 / 30 / 60 分钟）
- 离线收藏：已合成音频可缓存至本地

### 2.3 非功能性需求

| 需求类别 | 要求 |
|---|---|
| 性能 | 音频合成 P95 延迟 ≤ 30s；App 冷启动 ≤ 2s |
| 可用性 | 服务可用率 ≥ 99.9%；离线核心功能可用 |
| 安全 | 声音数据端到端加密；GDPR / 隐私法规合规 |
| 兼容性 | iOS 14+ / Android 8+；支持平板 |
| 无障碍 | 字体大小可调；适配孕期用户单手操作场景 |

## 三、用户体验设计原则

1. **温暖感**：主色调米白 + 粉橙，圆润字体，营造安全温馨感
2. **简洁操作**：核心功能 3 步内完成，减少孕期用户认知负担
3. **正向反馈**：每次录音给予积极的情感反馈
4. **家庭仪式感**：引导爸爸参与，强化「我们一起陪宝宝」的情感认同
